HTML Lists

Lists are used to group together related pieces of information so they are clearly associated with each other and easy to read. In modern web development, lists are workhorse elements, frequently used for navigation as well as general content. Lists are good from a structural point of view as they help create a well-structured, more accessible, easy-to-maintain document.

Two types Of Lists

  • Ordered Lists
  • Unordered Lists



Ordered Lists

Ordered (numbered) lists are used to display a list of items that should be in a specific order. An example would be list of computer components

  1. memory
  2. cpu
  3. monitor
  4. keyboard
  5. mouse


Unordered Lists

Unordered (bulleted) lists are used when a set of items can be placed in any order. An example is a moblie list: you

  • Samsung
  • Apple
  • Nokia
  • Sony
